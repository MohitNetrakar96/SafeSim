# NeonGlitch UI Template

A modern, high-performance web application template featuring:
- **React 19 + TypeScript**: The latest and greatest.
- **Tailwind CSS v4**: Next-gen styling with `@theme` based configuration.
- **Framer Motion**: Smooth, complex animations and transitions.
- **Glassmorphism Design**: Premium frosted glass aesthetics.
- **Dark Mode**: Optimized for visual comfort and style.

## Getting Started

1. **Install dependencies**:
   ```bash
   npm install
   ```
   *(Already done)*

2. **Run development server**:
   ```bash
   npm run dev
   ```

3. **Build for production**:
   ```bash
   npm run build
   ```

## Configuration

This project uses **Tailwind CSS v4**.

- **Theme Configuration**: Located in `src/index.css` inside the `@theme` block.
- **Custom Utilities**: Defined in `src/index.css` using `@utility`.
- **Layout Components**: `src/components/layout/`.
